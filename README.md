# Software destinado a Maleta Pedagógica 

## Blockly

Essa aplicação web foi desenvolvida de modo que pudesse ser utilizada de maneira estática nas maletas pedagógicas, tornando a conexão com a internet algo opcional, contudo pode ser executada em qualquer computador pessoal.

Utilizei a biblioteca Blockly do Google, de modo que pudéssemos fazer essa integração e utilizar as dependências 
do projeto que é open-source (código aberto - Significa dizer ser de livre acesso, assim como o nosso ).

### Equipe Envolvida

Tutoria: Professor Anselmo Costa

### Ferramentas utilizadas
- Blockly 
- Webpack
- HandleBars
- JavaScript
- HTML/CSS

#### Como Usar

- Instalar Dependências.

    > npm run install 

- Gerar Arquivos finais.

    > npm run build

- Executar Aplicação.

    > npm run start ( para executar através do servidor de desenvolvimento )

    ou 

    > Abra o arquivo "dist/index.html" diretamente no seu navegador
